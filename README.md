# React Server Side Rendering using Nashorn

- React
- Redux
- babel
- browserify
- Nashorn
- Spring Boot

# prepare

- install Node.js
- install yarn  
```
$ npm install -g yarn
```
- install npm modules
```
$ ./gradlew yarnInstall
```

# how to clean

```
$ ./gradlew clean
```

# how to run app

```
$ ./gradlew bootRun
```

# how to package

```
$ ./gradlew build
```
